# Гибридный поиск

**HYBRID_GAMMA_UNIQUE_8833**

Гибридный поиск комбинирует BM25 (разреженный) и векторный поиск для более точного извлечения.

## RRF (Reciprocal Rank Fusion)

RRF объединяет результаты нескольких поисковых систем:
```
score = sum(1 / (k + rank))
```
где k=60 — константа, rank — позиция в результатах.

## Режимы

- **fast** — только векторный поиск, быстро
- **quality** — гибридный (BM25 + вектор), точнее
