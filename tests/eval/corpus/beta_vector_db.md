# Chroma — Векторная база данных

**CHROMA_BETA_UNIQUE_5520**

Chroma — это встраиваемая (embedded) векторная база данных, оптимизированная для работы с эмбеддингами.

## Почему Chroma

Chroma работает локально, хранит данные в директории `chroma_db` и обеспечивает персистентность векторов. Это идеальный выбор для локальных RAG-систем.

## Основные операции

- **Добавление документов** с автоматической генерацией эмбеддингов
- **Поиск по similarity** — нахождение ближайших векторов
- **Фильтрация метаданных** — быстрый отбор по атрибутам
