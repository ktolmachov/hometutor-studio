### US-15.5 — Загрузить файл из UI для генерации карточек (P2)
**As a** Learner, у которого есть документ вне knowledge base,
**I want** загрузить PDF/TXT/DOCX прямо в интерфейс генерации,
**so that** я мог создать flashcards из любого учебного материала, не добавляя его в индекс.

**Acceptance:**
- Given я в разделе "Создать новые",
- When я переключаюсь на "Загрузить файл" и выбираю файл через `st.file_uploader`,
- Then файл отправляется на `/flashcards/generate` с `scope="upload"` и `content` (извлечённый текст).
- And flow генерации и preview идентичен сценарию с документом базы знаний.
- And колода сохраняется с `source_type="upload"` и именем загруженного файла.
