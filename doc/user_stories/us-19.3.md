# US-19.3 — Celebration UX (MoT #9 Graduation)

**Epic**: Epic 19: UX Breakthrough Wave  
**Priority**: P1  
**CJM Stage**: #9 Concept Graduation  
**Status**: open  
**Covered by**: ux-mastery-celebration-analytics (planned)

## User Story

**Как** Learner,  
**Я хочу** видеть яркое подтверждение завершения темы,  
**Чтобы** чувствовать достижение и мотивацию продолжать.

## Context

Graduation — это момент, когда learner достигает mastery ≥ 80% по теме. Это должен быть wow-момент, а не просто обновление числа в таблице. Celebration overlay с анимацией, метриками и следующим шагом превращает достижение в событие.

## Acceptance Criteria

### AC1: Graduation Trigger
**GIVEN** Learner достигает mastery ≥ 80% по теме  
**WHEN** система обнаруживает graduation  
**THEN** показывается Graduation_Overlay

### AC2: Celebration Animation
**GIVEN** Graduation_Overlay отображается  
**WHEN** overlay открывается  
**THEN** включается анимация celebration длительностью 2-3 секунды

### AC3: Achievement Badge
**GIVEN** graduation достигнут  
**WHEN** overlay показывается  
**THEN** система отображает Achievement_Badge с названием освоенной темы

### AC4: Key Metrics
**GIVEN** Graduation_Overlay активен  
**WHEN** метрики рассчитываются  
**THEN** overlay показывает: время обучения, количество сессий, финальный mastery%

### AC5: Badge Persistence
**GIVEN** Learner закрывает overlay  
**WHEN** overlay закрывается  
**THEN** система добавляет Achievement_Badge в профиль

### AC6: Next Topic CTA
**GIVEN** celebration завершена  
**WHEN** overlay закрывается  
**THEN** система предлагает Primary_CTA "Следующая тема"

### AC7: Dismissible Overlay
**GIVEN** Celebration_Overlay отображается  
**WHEN** Learner нажимает ESC или кликает вне области  
**THEN** overlay закрывается

## Definition of Done

- [ ] Graduation overlay triggers at 80% mastery
- [ ] Celebration animation 2-3 seconds
- [ ] Achievement badge persists to profile
- [ ] Next topic suggestion works
- [ ] ESC/click-outside dismiss works
- [ ] Unit tests для overlay components
- [ ] Property test для badge lifecycle
