### US-16.4 — Сгенерировать flashcards для курса (P0)
**As a** Learner,  
**I want** одним кликом сгенерировать flashcards по всем документам курса,  
**so that** у меня есть deck для повторения без ручной настройки каждого источника.

**Acceptance:**
- Given активирован StudyScope,
- When я перехожу в Flashcards и генерирую deck,
- Then карточки создаются по всем `source_paths` курса с тегами `course:<id>`.
- And карточки сохраняются и доступны в SRS review.
