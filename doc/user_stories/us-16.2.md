### US-16.2 — Запросы автоматически фокусируются на курсе (P0)
**As a** Learner с активным курсом,  
**I want** чтобы мои вопросы в Query tab автоматически искались по документам курса,  
**so that** мне не нужно вручную фильтровать каждый раз.

**Acceptance:**
- Given активирован StudyScope,
- When я задаю вопрос в Query tab,
- Then search выполняется только по документам активного курса (`folder_rel` фильтр).
- And вижу chip "Активный курс: [название]" над полем ввода.
