### US-20.1 — Получить объяснимую подсказку следующего учебного шага (P0)

**As a** Learner, который уже получил ответ, прошёл tutor/quiz или вернулся к прогрессу,
**I want** видеть один лучший следующий учебный шаг с коротким объяснением причины,
**so that** мне не нужно выбирать между режимами и я продолжаю полный learning loop без прокрастинации.

**Acceptance Criteria:**

- Given у пользователя есть learning state (последний ответ, trust signals, quiz results, due cards, mastery gaps или resume state),
- When он находится после RAG-ответа, tutor шага, quiz результата, flashcard review или на dashboard,
- Then UI показывает один primary next action с объяснением "почему сейчас".
- And подсказка детерминированно выбирается из состояния, а не выглядит случайной: например `answer_ready → Учить тему`, `quiz_failed → Разобрать слабое место`, `cards_due → Повторить карточки`, `mastery_stale → Проверить retention`.
- And у пользователя есть 2-4 безопасных secondary actions: открыть источники, объяснить проще, создать карточку, пропустить.
- And подсказка не ломает существующие entry points (`US-14.1`, `US-18.4`): home может показывать главный CTA, но Smart Study Router объясняет next action внутри всего учебного цикла.
