### US-20.12 - Quiet Mode Route

**As a** learner who is tired, distracted, or using assistive technology,  
**I want** a quieter router display,  
**so that** I can act on the same recommendation with less cognitive load.

**Acceptance Criteria:**

- Given quiet mode is enabled, the router still shows the primary action, reason, and alternatives.
- Given the card is rendered in quiet mode, keyboard and screen-reader order remains covered by fixture tests.
- Given quiet mode is disabled, the standard card remains unchanged.
