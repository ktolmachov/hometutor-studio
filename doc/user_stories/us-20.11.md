### US-20.11 - Micro-Outcome Receipt

**As a** learner,  
**I want** to see what changed after following the router recommendation,  
**so that** the next action feels useful rather than arbitrary.

**Acceptance Criteria:**

- Given a routed action completes, the UI shows a short local before/after receipt when measurable state changed.
- Given no measurable state changed, the receipt avoids fake progress and offers the next honest action.
- Given the learner returns to Progress, the receipt aligns with existing progress metrics.
