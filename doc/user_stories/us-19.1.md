# US-19.1 — Wait UX Improvements (MoT #2 First Answer)

**Epic**: Epic 19: UX Breakthrough Wave  
**Priority**: P1  
**CJM Stage**: #2 First Answer  
**Status**: open  
**Covered by**: ux-foundation-parsers-contracts + ux-first-answer-wait-flow (planned)

## User Story

**Как** Learner,  
**Я хочу** видеть прогресс во время ожидания первого ответа,  
**Чтобы** не чувствовать, что система зависла.

## Context

После достижения feature completeness (146 закрытых пакетов, все 13 MoT покрыты), UX составляет только 15.5% от всей работы. Следующий фокус — **perceived quality** и **wow-моменты** в ключевых точках пути пользователя. Wait UX — это первое впечатление от скорости системы.

## Acceptance Criteria

### AC1: Skeleton Screen
**GIVEN** Learner отправляет вопрос  
**WHEN** система начинает обработку  
**THEN** Skeleton_Screen появляется в течение 100ms

### AC2: Progress Indicator
**GIVEN** система генерирует ответ  
**WHEN** обработка идёт  
**THEN** индикатор прогресса показывает текущий этап (Retrieving sources... / Generating answer...)

### AC3: Progressive Reveal
**GIVEN** первые фрагменты ответа готовы  
**WHEN** LLM начинает генерацию  
**THEN** система начинает Progressive_Reveal контента

### AC4: Perceived Latency
**GIVEN** любой вопрос  
**WHEN** система обрабатывает запрос  
**THEN** Perceived_Latency не более 2 секунд до первого видимого контента

### AC5: Typewriter Effect
**GIVEN** ответ формируется  
**WHEN** контент раскрывается  
**THEN** система применяет Typewriter_Effect со скоростью 50-80 символов/секунду

### AC6: Layout Consistency
**GIVEN** Skeleton_Screen отображается  
**WHEN** финальный контент загружается  
**THEN** структура skeleton соответствует финальному layout

## Definition of Done

- [ ] Skeleton screen рендерится <100ms
- [ ] First content visible <2s
- [ ] Typewriter effect 50-80 chars/sec
- [ ] AnswerObject contract доступен для финального отображения
- [ ] Unit tests для Wait UX компонентов
- [ ] Integration test для end-to-end flow
- [ ] Performance benchmarks в пределах целевых значений
