### US-20.9 - Learning Debt Queue

**As a** learner,  
**I want** the router to distinguish memory debt, weak concepts, and new learning,  
**so that** I can understand whether the next step protects retention or advances the course.

**Acceptance Criteria:**

- Given due cards exist, the router can label the action as retention debt with a local count or age reason.
- Given a quiz failure is active, the router can label the action as weak-concept recovery.
- Given neither retention nor recovery signals dominate, the router can label the action as new learning or continuation.
