### US-17.2 — Course Cockpit (P0)

**As a** Learner в активном курсе,  
**I want** единый экран-"кабина" для всех активностей курса,  
**so that** я не переключался между табами и не терял контекст и поток.

**Acceptance:**
- Given у меня есть активный StudyScope,
- When я открываю Course Cockpit (entry из home_hub или Topics),
- Then вижу трёхколоночный layout: Path Map (слева) | Active Activity (центр) | Live Progress (справа).
- And центральная колонка auto-advances: cards → micro-quiz → tutor → scenario без ручной навигации.
- And left Path Map показывает все концепты курса с визуальным статусом (gray/learning/yellow/green/gold).
- And right колонка показывает live counters (mastered, due, streak, ETA) + "next up" список.
- And persistent header: course name, pace selector, pomodoro timer, кнопка Exit.
- And при feature flag `RAG_COURSE_COCKPIT_V2=0` классический tab-flow сохраняется без изменений.

**New modules:** `app/ui/course_cockpit.py`  
**Reuses:** `study_scope`, `continuity_bridge`, `flashcards_sections`, `quiz_panel`, `tutor_orchestrator`  
**Metric:** tab-switches per session (target 0), avg session duration (target ≥18 min)
