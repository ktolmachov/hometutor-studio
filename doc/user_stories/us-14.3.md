### US-14.3 — Быстро открыть expert controls без потери beginner path (P1)
**As an** experienced Learner / maintainer,
**I want** иметь компактный advanced layer для retrieval profile, filters, router trace, eval/debug, graph view и export/import,
**so that** я могу быстро диагностировать и управлять системой, не превращая первый экран в cockpit.

**Acceptance:**
- Given я хочу проверить retrieval/debug/graph детали,
- When я открываю advanced/expert controls,
- Then retrieval profile, filters, router trace/debug, graph controls и export/import остаются достижимыми.
- And по умолчанию beginner entry не показывает raw trace или eval controls как основные действия.
- And закрытие advanced layer возвращает learner к guided CTA без потери состояния.
