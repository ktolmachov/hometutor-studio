# Глоссарий home-rag

Короткие определения для пользовательского гайда и roadmap-документов.

| Термин | Что означает |
|---|---|
| SSR / Smart Study Router | Умный маршрутизатор, который выбирает один следующий учебный шаг из локальных сигналов: карточки, квизы, план, граф знаний и незавершённые сессии. |
| SM-2 | Алгоритм интервального повторения: оценка ответа управляет тем, когда карточка появится снова. |
| Due-очередь | Карточки или темы, которые уже пора повторить. |
| Mastery vector | Локальная карта сильных и слабых концептов пользователя. |
| Graduated | Тема считается стабильно освоенной и меньше мешает текущему плану. |
| Knowledge Graph | Граф тем и связей между ними, включая зависимости и соседние понятия. |
| Rule-based fallback | Безопасная базовая логика без модели; используется при сбое, нехватке данных или выключенном AI-пути. |
| Cold-start gate | Порог данных, до которого ML-компонент не включается в serving и система остаётся на правилах. |
| Eval contract | Проверочный контракт перед AI/ML-фичей: метрики, датасет, fallback и критерии остановки. |
| Hybrid Intelligence | Подход, где правила остаются страховочной сеткой, а ML/LLM улучшает объяснение, ранжирование или планирование. |
