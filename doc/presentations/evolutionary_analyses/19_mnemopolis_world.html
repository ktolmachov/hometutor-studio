1Суть

Чего по-настоящему хочет ученик, гуляющий по графу своих знаний

Не «поиграть» — увидеть. У hometutor уже есть всё, что нужно для честного ответа на вопрос «что происходит с моей памятью»: граф, spaced repetition, история квизов, RAG-тьютор, паспорт качества конспекта. Но это знание живёт в цифрах и панелях, а человек мыслит образами и местом. 3D-зал был первым органом чувств для этого знания — и он сработал (владелец: «Вау!» после Memory Run). Дальше вопрос не «добавить ли ещё фич», а «может ли зал стать единственным честным зеркалом всего продукта» — тем местом, куда возвращаются между разделами, а не тем, что открывают раз и забывают. Это и есть суть разбора: 3D — не украшение поверх продукта, а орган чувств, которого раньше не было ни у одного его раздела.

3Противоречия

Напряжения — и их синтез

Local-first, ноль CDN ↔ «мировой класс» визуала

Продукт принципиально offline (провайдер-слой, вендоринг d3/mermaid по тому же правилу, что и здесь). «Мировой класс» обычно читают как Three.js/WebGL/частицы. Синтез: глубина — через семантику цвета и честное движение, не полигоны. Reference Memory Run уже доказал: canvas 2D + свет + пунктир держат уровень «Вау» без единой внешней зависимости. Кандидатура WebGL остаётся закрытой, пока canvas 2D не упёрся в потолок выразительности — а он пока не упёрся.

ИИ-хранитель мира ↔ заморозка домена

Расширение мира просит агента, который «достраивает, объясняет, управляет» — но kill switch v1/v2 запрещал LLM в 3D-зале целиком. Оставить запрет — мир остаётся немым; снять его без границ — риск, что текст модели незаметно начнёт решать за систему, что ученик «знает». Синтез: LLM разрешён строго как рендер текста/сцены поверх уже посчитанных данных — provider layer, кэш по (snapshot_date, concept_id, сценарий), бюджет вызовов, обязательная офлайн-деградация, и жёсткое «пишет в домен» = запрещено по построению (scene-DSL — whitelist презентационных команд, не мутаций mastery/SR/workbench).

Мотивация ↔ анти-паттерн «вторая валюта»

Игровой слой должен поднимать желание пройти путь — а любимый инструмент геймдизайна тут прямо запрещён и предыдущими разборами, и здравым смыслом (XP-инфляция, нездоровые петли вовлечения не место в обучающем продукте). Синтез: мир не изобретает валюту — визуализирует уже существующую (gamification_service.py: XP/уровень/стрик/бейдж). Рассвет неба, высота Обсерватории, свет фонарей — 100% derived от того, что уже считается; ноль новых чисел.

Опасность как двигатель внимания ↔ «не пугать, не стыдить»

Антагонисты — сильный крючок вовлечения в любой игре, но hometutor — учебный продукт, а не аркада; тревога — плохой педагогический инструмент. Синтез: угроза = приглашение, не приговор. Ровно три антагониста, каждый — честная визуализация реального врага обучения (забывание, ложная уверенность, пропущенный пререквизит), ни один не блокирует контент, каждый закрывается действием ≤2 минуты, есть тумблер «Спокойный мир», и ни один не показывается в первом route-кадре (ориентация №17 неприкосновенна).

4Боль-якорь

Гейт стал зорче — но по-прежнему верит на слово двум разным художникам

Проверяемый факт: ось «СЕГОДНЯ» больше не перекрыта кнопками навигации (закрыто 271–274) — но координата, на которой она рисуется, и высота панели, от которой она отстраивается, живут в двух разных файлах как два независимых литерала, никогда не сверяемых друг с другом ни в коде, ни в тесте.

Точная причина в коде:

  • app/ui/assets/kg_3d_template.html:126 — DOM-панель .kgx-route-actions закреплена CSS-константой bottom:4.6rem/аналогичной раскладкой, независимо считанной для мобильного брейкпоинта (:354 — иной набор чисел для узких экранов);
  • app/ui/assets/kg_3d_template.html:1270 — canvas отдельно вычисляет axisY = Math.max(H * 0.86, H - 78): число 78 — это рукописная оценка высоты DOM-панели сверху, а не getBoundingClientRect() этой самой панели;
  • у шаблона уже есть собственная система защиты от коллизий для canvas-подписей узлов — labelBoxes/labelIntersects (:1008-1013, используется drawSmartLabel и drawActiveReasonCallout) — но ось «ВХОД/СЕГОДНЯ/ДАЛЬШЕ» в эту систему не зарегистрирована и не резервирует зону DOM-панели как «занятую»;
  • tests/test_knowledge_graph_counters.py:855-874 — новый (271–274) geometry-gate меряет высоту топбара, ширину панели, число иконок, переполнение body — но ни один assert не читает axisY и не сверяет его с .kgx-route-actions.getBoundingClientRect().

Житейски: сегодняшний фикс — это не «мы научили сцену не сталкиваться», а «мы один раз руками подобрали число 78, которое сегодня совпадает с реальной высотой панели». Стоит панели вырасти на пиксель (новая иконка, локализация с более длинным словом, изменённый брейкпоинт) — число 78 не узнает об этом первым; узнает пользователь, когда снова увидит текст под кнопкой. Тот же урок, что нашёл прошлый аудит на пустом 1×1 canvas №17: не «баг закрыт», а «класс бага не закрыт», пока два слоя сцены не измеряют друг друга, а угадывают.

Решение по каждому элементу мира

оставитьCanvas 2D, не WebGL/Three.js. Reference Memory Run уже доказал достаточность на production-данных; смена движка — решение, отдельное от этого разбора, не обязательное условие «мирового класса».

оставитьСкорость закрытия аудита (271–274, тот же день). 9 находок → geometry-gate + фикс за один день — образцовый темп, не трогать процесс, которым это было сделано.

починитьНазвание «3D-зал» → «Мнемополис». Чисто UI-переименование поверх того же render-контракта; ноль доменных изменений, ноль новых схем.

6Ценность

Уникальность — и измеримое доказательство

У геймификаторов обучения есть один общий приём: нарисовать талисман (сова, дракон, питомец) поверх честной статистики прогресса — картинка живёт отдельно от данных и ничего не теряет, если данные исчезнут. Мнемополис устроен наоборот: картинка и есть данные. Нет тумана — нет декоративного слоя, есть только показание decay_vector. Нет рассвета — нет анимации, есть только доля пройденного маршрута. Убери любой объект мира — под ним всегда обнажится реальное число продукта. Это сочетание — граф знаний + spaced repetition + история квизов + RAG + локальный ИИ, честно спроецированные в одну сцену — уникально для этого приложения и не воспроизводится косметическим редизайном конкурента.

7Путь

Золотой путь и лестница уровней мира

Золотой путь одной сессии (после P0): открыл Мнемополис → небо уже теплее вчерашнего (рассвет по вчерашнему прогрессу) → тропа дня без единой коллизии геометрии, читается за 5 секунд → тронул подёрнутую туманом остановку → 2 карточки spaced repetition → фонарь зажёгся, туман отступил → «Дальше» → квиз → вернулся в зал с трофеем, тропа за спиной светится длиннее.

  1. 0Скин (сегодня). Memory Run визуально узнаваем, геометрия исправна на живом экране — но чинится вручную, не измерением; следующая визуальная регрессия того же класса ничем не гарантированно поймана.
  2. 1Измерение + Рассвет. Zero-drift геометрия (DOM меряет сам себя для canvas); небо и фонари делают прогресс дня видимым без чтения панели — первый настоящий «Вау» уровня мира, не только скина.

  4. 3Хранитель заговорил. Экскурсовод и сводка угроз — первые LLM-сценарии, строго по контракту §3/§6 vision-документа.
  5. 4Районы и порталы. Каждый раздел приложения — дверь в мире и обратно; «🌆 в Мнемополис» — кнопка на каждой странице.
  6. 5Собеседник и NL-командование. «Спросить об этом» прямо в интерьере узла; голосовая команда сцены через scene-DSL.
