## Примеры правильного использования

### ✅ Правильно:

```markdown
**Уровень 1: Локальное машинное обучение — Модель кривой забывания**

**Решение:** ML-модель предсказывает оптимальное время повторения

**Технически:** RAG-поиск → генерация через LLM с указанием источников
```

### ❌ Неправильно:

```markdown
**Уровень 1: Локальное МО — Модель кривой забывания**

**Решение:** МО-модель предсказывает оптимальное время повторения

**Технически:** Поиск с дополненной генерацией → генерация через ЯМ
```
