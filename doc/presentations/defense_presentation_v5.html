03 Проблема и решение

Куча документов → ценность?

Проблема

  • Универсальный чат не знает ваших документов
  • Поиск по файлам даёт фрагменты без контекста
  • Ручные Anki-карточки — долго и рутинно
  • Непонятно, что освоено, а что нет

Следствие: переключение между инструментами и потеря контекста.

Решение — один инструмент, полный цикл

Документы → Q&A → Тьютор → Квиз → Карточки → Mastery → Graduation ↑ Smart Study Router (подсказка следующего шага)

Local-first: данные и модель — на машине пользователя.

04 Полный учебный цикл

Ответ — это первый шаг, а не финал

В типичной RAG-системе сценарий заканчивается на ответе. Здесь — начинается.

1–2 · Ответ → Тьютор

RAG-ответ с источниками и confidence; объяснение наводящими вопросами, а не лекцией.

3–4 · Квиз → Карточка

Проверка понимания; важное → SM-2 карточка без копипасты.

5–6 · Повторение → Mastery

Интервальное повторение; recognition → recall → transfer → graduation.

07 Smart Study Router · AI Vision

«Много режимов — что делать дальше?»

Deterministic-first: считает локальные сигналы, выбирает следующий шаг, объясняет «почему сейчас». AI/ML — как gated enrichment, не как обязательная зависимость.
УровеньЧто добавляетСтатус
L1 · локальная памятьML forgetting curve, reranking (AUC-ROC ≥ 0.75)доставлен; serving после real samples
L2 · объясненияLLM-объяснения + semantic caching (p95 < 2с)доставлен
L3 · недельный планrule-based weekly planner + telemetryдоставлен
L4 · граф prerequisitesprerequisite-aware routingдоставлен (feature-flag)
L5 · обучение на отказахmisroute policy learning (offline, decay)доставлен

Базовая маршрутизация объяснима и работает без облачного профиля пользователя.

08 Доверие и заземление

Grounding Contract: факт ↔ источник

Механизмы

  • Inline-цитаты [N] — факт ссылается на фрагмент
  • Provenance ledger — факт → источник → chunk
  • Honest abstain — при слабом контексте отказ, не догадка
  • Citation tolerance — лишние/недостающие ссылки не ломают ответ
  • Confidence — видно, насколько ответ обоснован

Трёхслойная оценка качества

+ RAGAS-метрики (context_precision@k, answer_correctness) поверх eval-harness.

10 AI-фабрика разработки

«Документ — закон, скрипт — движок, агент — руль»

Продукт делается не вручную, а воспроизводимым AI-конвейером. Миграция выделила фабрику в репозиторий hometutor-studio.

Backlog registry (SSoT)

backlog_registry.yaml — единственный источник статусов; tasklist.md производный.

Командный конвейер

PO → Analyst → Architect → Dev → Tester как воспроизводимый процесс.

Token-safety registry

Предсказуемый бюджет контекста (≤12k) для каждого LLM-вызова.

Локальный coding-trigger

llama.cpp как patch-executor: diff → WRITE_SET gate → git apply → targeted tests.

Evidence-gated поставка: пакет закрывается только при пройденных DoD-тестах и обновлённых docs. Kill-switches блокируют filename-fallback за semantic-граф, связи без evidence, graph-aware без uplift-gate, скрытый cloud-вызов в local-профиле.
