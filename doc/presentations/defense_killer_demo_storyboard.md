## 📱 Адаптация для разных форматов

### Для проектора (16:9)

- Увеличить шрифты: минимум 18pt
- Больше whitespace между элементами
- Яркие контрастные цвета

### Для записи видео

- Добавить voice-over с музыкой
- Замедлить transitions для clarity
- Добавить subtitles для ключевых моментов

### Для печатного handout

- Статичные скриншоты всех 8 кадров
- QR-код на видео-версию
- Краткие подписи под каждым кадром
