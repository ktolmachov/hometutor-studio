## 🎬 Открытие (5 сек)

> "За три минуты покажу **одну учебную историю**: от вопроса до персонального плана. Вопросы после демо."

## 🎯 Демо-вопрос (заготовка)

```
Объясни, почему RAG-ответ сам по себе недостаточен для обучения, 
и как связка tutor, quiz, flashcards и mastery plan закрывает 
этот разрыв. Укажи источники.
```

## 💬 Ключевые реплики

### Кадр 1 (20 сек)
> "Начнём с обычной ситуации: учебные материалы — PDF, markdown, заметки. Система проиндексировала их **локально** — никакого облака."

### Кадр 2 (35 сек)
> "Вопрос специально **не фактологический**. Требует синтеза из нескольких источников. Именно такие вопросы задают студенты."

### Кадр 3 (25 сек)
> "Главное не текст, а то, **на чём он стоит**. Три источника, кликабельные. Retrieval confidence показывает покрытие. Раскрываю первый источник."

### Кадр 3.5 — 🧭 Умный помощник (20 сек) — KILLER FEATURE!
> "А вот **killer-фича** — **Умный помощник**. Система не просто показывает кнопку, а **объясняет, почему именно сейчас** это лучший шаг. Видите: 'Локально уже есть быстрый ответ — перенесите материал в учёбу'. Это решает когнитивную перегрузку выбора — студент не тратит 2-3 минуты на раздумья. Детерминированная логика без LLM-вызовов."

### Кадр 4 (20 сек)
> "Нажимаю 'Учить тему' (по рекомендации Умного помощника) — система переходит из assistant в персонального учителя."

### Кадр 5 (25 сек)
> "Tutor **вытаскивает понимание через вопросы**. Сократический метод. Отвечаю: 'Нужна проверка и повторение'. Система подтверждает."

### Кадр 6 (20 сек)
> "Квиз проверяет **понимание**, не красоту ответа. Момент, где assistant становится учебным инструментом. Mastery +5%."

### Кадр 7 (15 сек)
> "Карточка создаётся **без копирования**. Система запланировала повторение на завтра. Автоматический переход к запоминанию."

### Кадр 8 — Финал (30 сек)
> "Система обновила модель знания и предложила **персональный план на завтра**. За три минуты: материал → вопрос → ответ → **умный помощник объяснил следующий шаг** → понимание → проверка → карточка → план. **Полный learning loop с умным гидом, а не чат с файлами.** Единственная система, которая **объясняет следующий шаг**. Спасибо, готов к вопросам."

## 🎯 Что доказываем

| Кадр | Доказательство |
|---|---|
| 1-2 | Работает с реальными материалами пользователя |
| 3 | Ответ **проверяемый**, не магический |
| **3.5** | **Умный помощник объясняет "Почему сейчас" — KILLER FEATURE!** |
| 4 | Переход от Q&A к **учебному циклу** |
| 5 | Есть **педагогическая методика** |
| 6 | Есть **контроль знания** |
| 7 | Переход к **запоминанию** автоматизирован |
| 8 | **Замкнутый цикл** обучения с умным гидом |

## 🛡️ Если что-то пошло не так

| Проблема | Решение |
|---|---|
| LLM медленно отвечает | "Обычно быстрее, показываю логику" |
| Ответ неожиданный | Использовать fallback-вопрос |
| UI завис | Перейти на backup-скриншоты |
| Вопрос из зала | "Отвечу после демо, чтобы не потерять нить" |
