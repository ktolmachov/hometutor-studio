## RAG Pipeline

**5-ступенчатая обработка запроса**

```
CLASSIFY → REWRITE → RETRIEVE → RERANK → GENERATE
  Router    Enricher   Hybrid     Cross-    LLM +
  qa/over   улучшение  BM25+Vec   encoder   sources
```

**Инновации:**
- Classify определяет тип запроса
- Rewrite улучшает формулировку
- Retrieve использует гибридный поиск
- Rerank точно оценивает релевантность
- Generate создает ответ по источникам
