# Круглые столы и панельные дискуссии

Симулированные панельные обсуждения hometutor по итогам серии эволюционных разборов
№1–№10 ([`../evolutionary_analyses/`](../evolutionary_analyses/)). Жанр отличается от
разборов: не «суть → боль-якорь → план», а голоса разных аудиторий вокруг уже
поставленных диагнозов.

**Правила жанра (зафиксированы при создании, 2026-07-12):**

1. **Персонажи вымышлены, факты — нет.** Каждое утверждение о продукте в репликах
   проверяется по коду на актуальном HEAD и снабжается подписью `файл:строка`
   (класс `.ev` под репликой). Панель без фактчека — фанфик, не документ.
2. **Стиль — notebook v1.0** ([`../notebook_deck_guide.md`](../notebook_deck_guide.md)):
   копия `notebook_template.html`, плюс жанровый компонент `.say`/`.who`
   (реплика с подписью; вариант `.say.heat` с красной кромкой — для троллей,
   `.say` с зелёной — для пользователей). Палитра не меняется.
3. **Честная рамка в `meta`:** каждая страница явно помечает «формат: симулированная
   панель — персонажи вымышлены».
4. **Каждый стол заканчивается синтезом**, который сверяется с действующим планом
   (`../../next/full_circle_visibility_plan.md`) — панель ценна тем, что независимыми
   голосами подтверждает или корректирует приоритеты.
