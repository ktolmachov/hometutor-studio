## 📋 Правила применения

1. **Заголовки слайдов** — переводить полностью для понятности
2. **Технические термины в тексте** — оставлять RAG, LLM, ML, embeddings
3. **Код** — не трогать вообще
4. **Ссылки** — не трогать
5. **Названия компонентов** — переводить для понимания (Router → Маршрутизатор)
6. **Метрики** — переводить описание, оставлять технические названия в скобках
7. **Таблицы** — переводить заголовки и описания, оставлять технические термины

## 🎯 Целевая аудитория

**Кто будет слушать:**
- Преподаватели без глубокой технической подготовки
- Студенты, не являющиеся экспертами в ИИ
- Потенциальные пользователи продукта
- Эксперты, которые оценивают практическую ценность

**Что важно:**
- Понятность без потери точности
- Сохранение технических терминов там, где они устоялись
- Объяснение сложных концепций простыми словами
- Баланс между доступностью и профессионализмом
