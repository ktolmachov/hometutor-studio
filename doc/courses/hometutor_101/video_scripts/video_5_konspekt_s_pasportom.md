## Примечания для генерации

- Голос: ~120 слов; ключевые паузы на кадрах 2 и 4.
- Кадры, помеченные `*`, в витрине ещё не сняты (фичи паспорта и статусов
  отгружены недавно): снять по конвейеру Playwright новые манифесты или,
  до съёмки, использовать живой скринкаст этих экранов. **Мокапы запрещены** —
  правило витрины: только реальные кадры продукта.
- Кадры не редактировать содержательно; только зум/панорама/подсветка.
