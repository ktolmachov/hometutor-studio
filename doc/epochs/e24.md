## E24-A: Learner Goal Contract / Learning Kernel kickoff (2026-04-12)

- **CJM / US:** Tutor session / Learn; **US-14.4** (goal в 5-минутном цикле), **US-6.1** (next-step / plan совместимость), **US-9.1** (компактный fallback без дублирования Progress). Полный Learning Kernel (персистентность целей, план-сессия, recap) — вне scope.
- **Backend:** опциональные поля `tutor_goal_*` в `AskRequest` / `QueryOptions`; `prepare_ask_request` нормализует строки (≤512) и минуты 1–240; `build_tutor_session_state` заполняет стабильный `learner_profile["goal_context"]` и `learner_goal_context` в metadata tutor-ветки `answer_question`. Новых endpoints, LLM, миграций SQLite нет.
- **UI:** session keys `tutor_goal_*`; `_tutor_query_options` передаёт поля в backend; «Сейчас: …» через `e24_active_goal_line_ru`; после мини-проверки в 5-минутном цикле — `e24_five_min_closure_combined_ru` (E11 weekly goals + опциональная строка цели); Q&A CTA «Учить эту тему 5 минут» задаёт бюджет 5 мин и subtopic/outcome.
- **Тесты:** `tests/test_e24_goal_context.py`, `tests/test_e24_goal_ui.py`, регресс `test_e11_learning_loop`, `test_models`, `test_api`, `test_tutor_orchestrator`.
- **Артефакты:** `archive/team_artifacts/E24-A/` (PO → closure).

## E24-B: Learner goal snapshot persistence (2026-04-12)

- **CJM / Retain:** снимок цели между сессиями (без полного Learning Kernel).
- **SQLite:** таблица `learner_goal_snapshot` (id=1), `schema_version`, поля в форме `goal_context` (E24-A); участвует в `export_full_sync_bundle` / `import_full_sync_bundle` через `_SYNC_TABLES_ORDER`.
- **HTTP:** `app/routers/learner.py` — `GET/PUT/DELETE /learner/goal-snapshot`; модели `LearnerGoalSnapshotPutRequest`, `LearnerGoalSnapshotOut` / `LearnerGoalContextOut`; регистрация в `app/api.py`.
- **Acceptance (tasklist):** round-trip через `user_state` после «перезапуска»; первое обращение к API на пустой БД — схема создаётся, GET не падает (`goal_context: null`).
- **Follow-up:** подстановка в Streamlit / `POST /ask` — см. **E24-B-2** ниже (закрыто 2026-04-12).
- **Проверки:** `pytest tests/test_user_state.py tests/test_e24_goal_persistence.py -q`.

## E24-B-2: Persisted goal → `POST /ask` + Streamlit (2026-04-12)

- **CJM / US:** Retain + Tutor session; снимок **E24-B** подставляется в запрос и в UI без ручного копирования.
- **Backend:** `app/ask_goal_snapshot_merge.py` — перед `prepare_ask_request` в `app/routers/query.py` подмешиваются незаполненные `tutor_goal_*` из `get_learner_goal_snapshot()`; явные поля тела запроса имеют приоритет.
- **UI:** `hydrate_tutor_goal_snapshot_once()` + `persist_tutor_goal_snapshot_from_session()` в `app/ui/session_state.py`; вызов hydrate из `app/ui/main.py`; persist после CTA «Учить эту тему 5 минут» (`query_tab.py`), после onboarding и сценариев входа на home (`home_hub.py`); «Сбросить цель сессии» — `clear_tutor_goal_and_snapshot()` (SQLite + session keys + сброс флага гидратации).
- **Документация:** `doc/api_reference.md`, `doc/changelog.md`, `doc/tasklist.md`.
- **Проверки:** `pytest tests/test_api.py tests/test_e24_goal_context.py tests/test_user_state.py tests/test_e24_goal_persistence.py tests/test_e24_goal_ui.py -q`.
