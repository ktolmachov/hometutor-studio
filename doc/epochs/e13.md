## E13: Home Mode Selector / UX Tail (2026-04-12)

- **Owner decision:** deferred `17.1 UX tail` оформлен как закрытый epoch-package E13-A после фактического редизайна главного экрана.
- **CJM / US:** Discover → First Answer → Learn → Retain → Progress; переиспользованы **US-14.1**, **US-7.3**, **US-15.2**, **US-9.1** без добавления новой user story.
- **UI:** `render_mode_selector()` в `app/ui/home_hub.py` стал основным навигационным элементом home: 2 ряда × 3 карточки (`Тьютор`, `Быстрый ответ`, `Quiz`, `Flashcards`, `Темы`, `Прогресс`) + collapsed `Ещё инструменты`.
- **Declutter:** Quiz Hero, Adaptive Daily Plan Hub, 3-колоночная строка Resume/Due/Flashcards, mastery gauge, полный Adaptive Daily Plan expander, entry navigation buttons и XP/streak/level bar убраны с главного экрана; целевые поверхности остались в своих вкладках.
- **Интеграция:** due badge на Flashcards берёт `/flashcards/due/count`; карточки меняют `st.session_state["current_view"]` и вызывают `st.rerun()`, selectbox ниже подхватывает выбранный view.
- **Guardrail:** без изменений router/eval/tutor-runtime; responsive проверен для 900px и 640px по итоговому UX-scope.
