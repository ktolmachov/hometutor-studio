## E10.3: Router Eval Baseline (2026-04-11)

- **US-12.4:** `app/router_eval.py` — вызов `invoke_pedagogical_orchestrator_llm` vs `router_eval.gold_selected_agent`; `eval_data/tutor_regression.json` v1.3 — **26** кейсов с gold; опционально `stub_user_message` если нет `input.question`.
- **Скрипт:** `scripts/run_router_eval.py` — JSON с `aggregate` (overall + per-category для отчёта), `--baseline eval_data/router_eval_baseline.json`; **регрессия только по `overall_accuracy`** (строгое `delta < -5/100` при default threshold, т.е. −5.00 п.п. ещё допустимо); per-category gate нет. **`--limit`** на неполном наборе отключает сравнение с baseline (smoke не смешивается с full baseline). - **Проверки:** `pytest tests/test_router_eval.py` (мок LLM).
