# Векторные хранилища

Chroma — пример embedded vector database для локального хранения эмбеддингов.
Коллекции группируют документы; метаданные позволяют фильтровать по файлу и папке.
Для продакшена важны бэкапы каталога chroma_db и совместимость модели эмбеддингов.

Уникальный якорь beta: **CHROMA_BETA_UNIQUE_5520**.
