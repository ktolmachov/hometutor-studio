# Введение в RAG

Retrieval-Augmented Generation (RAG) объединяет поиск по базе знаний и языковую модель.
Ключевая идея: перед генерацией ответа система извлекает релевантные фрагменты документов.
Типичный пайплайн: вопрос → эмбеддинг → векторный поиск → подсказка в промпт LLM.
