# Defense Presentation - Quick Start

## 🎯 Что это?

Академическая презентация проекта home-rag_v2 в формате PDF.

- **14 слайдов** в формате 16:9
- **2.47 MB** - оптимизированный размер
- **Векторный текст** - TrueType шрифты
- **Готово для импорта** в Google Slides, PowerPoint, Keynote

## ⚡ Быстрые команды

### Сгенерировать презентацию

```bash
npm run docs:defense-pdf:visual
```

### Проверить качество

```bash
npm run docs:defense-pdf:check
```

## 🎨 Использование

### Для защиты
Откройте `doc/defense_presentation.pdf` и используйте как основу для выступления.

### Для импорта
Импортируйте в Google Slides, PowerPoint или Keynote для редактирования.

### Для печати
Распечатайте в формате 16:9 для раздаточных материалов.
