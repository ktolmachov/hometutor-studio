# Implementation Plan: project-defense-materials

## Overview

Создание двух Markdown-документов для академической защиты проекта **home-rag_v2** и набора тестов для их верификации. Реализация выполняется в три шага: скрипт выступления → презентация → тесты. Каждый шаг опирается на реальные артефакты репозитория; перед записью любой ссылки на файл — проверить существование через `listDirectory`.
